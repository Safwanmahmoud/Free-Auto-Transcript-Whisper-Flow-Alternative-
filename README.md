# Free-Auto-Transcript-Whisper-Flow-Alternative

## Description
This is a free alternative for those who prompt a lot and prefer talking to typing to save time. just press ```Ctrl + Win``` and talk.

## How to use 
1. Clone the repo (or just download files).
2. Add .env file and add (GOOGLE_API_KEY=" you api key "), which you can get from [Google AI Studio](https://aistudio.google.com/app/api-keys?project=gen-lang-client-0317836474) for free.
3. Create a venv and install requirements.txt
4. Press Win + R and type ```shell:startup``` it will open the startup folder.
5. Right-click and choose new -> shortcut. Navigate to the downloaded repo and point at "run_startup.bat".
6. Restart your PC. It will work on the background after.
